# 前言

您好，欢迎来到基于SSM的校园心理辅导系统项目仓库。此项目旨在为我国校园提供一套全面、高效的心理辅导解决方案，通过信息化手段提高心理辅导工作的质量和效率。

# 内容介绍

本系统采用Java语言，结合Spring、SpringMvc和Mybatis框架进行开发，前端使用JS、Vue和CSS3技术。系统主要包括学生信息管理、预约咨询、心理测评、咨询记录管理等功能模块，致力于为学校心理辅导工作提供便捷、智能的服务。

# 技术介绍

- **语言：** Java
- **使用框架：** Spring、SpringMvc、Mybatis
- **前端技术：** JS、Vue、CSS3
- **开发工具：** IDEA/Eclipse
- **数据库：** MySQL 5.7/8.0
- **数据库管理工具：** phpstudy/Navicat
- **JDK版本：** jdk1.8
- **Maven：** apache-maven 3.8.1-bin
- **前端环境：** Node.Js 12\14\16
